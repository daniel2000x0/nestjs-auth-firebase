## 🚀 Description

This is a simple [NestJS](https://nestjs.com/) starter project that demonstrates:

- Integration with **Firebase Admin SDK**
- Basic **authentication** using Firebase
- RESTful API setup with modular structure

Useful for quickly bootstrapping NestJS applications that rely on Firebase services like Firestore, Auth, and more.
